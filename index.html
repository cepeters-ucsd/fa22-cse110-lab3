What is the game?

Loop Arena is a game created for Ludum Dare 51. I made it the night before/morning of the end of the jam, so it's pretty simple, but it's a project I had ready for this meeting minutes. It's a simple arcade game where you fight enemies that spawn in a 10-second loop, with each loop getting progressively harder.
